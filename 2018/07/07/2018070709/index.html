利用一阶谓词逻辑求解猴子摘香蕉问题

7月 07, 2018 发布在 算法

  利用一阶谓词逻辑求解猴子摘香蕉问题：房内有一个猴子，一个箱子，天花板上挂了一串香蕉，其位置如图1所示，猴子为了拿到香蕉，它必须把箱子搬到香蕉下面，然后再爬到箱子上。请定义必要的谓词，列出问题的初始化状态（即图1所示状态），目标状态（猴子拿到了香蕉，站在箱子上，箱子位于位置b）。

图1 猴子摘香蕉

1 实验方案

1.1 定义谓词

谓词描述了具体的性质和状态，本题为描述猴子摘香蕉这一问题，定义了以下几个谓词：

1) $AT\left( x,w \right)$：表示$x$在$w$处。
其中个体域：$x\in \left\{ monkey \right\}$，$w\in \left\{ a,b,c \right\}$。

2) $HOLD\left( x,t \right)$：表示$x$拿着$t$。
其中个体域：$t\in \left\{ box,banana \right\}$。

3) $EMPTY\left( x \right)$：表示$x$手中是空的。

4) $ON\left( t,y \right)$：表示$t$在$y$处。
其中个体域：$t\in \left\{ b,c,ceiling \right\}$。

5) $CLEAR\left( y \right)$：表示$y$上是空的。

6) $BOX\left( u \right)$：表示箱子$u$。
其中个体域：$u\in \left\{ box \right\}$。

7) $BANANA\left( v \right)$：表示香蕉$v$。
其中个体域：$v\in \left\{ banana \right\}$。

1.2 利用谓词、联结词和量词来表示环境状态

1) 初始状态

S0：$AT\left( monkey,a \right)$^$EMPTY\left( monkey \right)$^$ON\left( box,c \right)$^$ON\left( banana,ceiling \right)$^$CLEAR\left( b \right)$^$BOX\left( box \right)$^$BANANA\left( banana \right)$

2) 目标状态

Sg：$AT\left( monkey,a \right)$^$HOLD\left( monkey,banana \right)$^$ON\left( box,b \right)$^$CLEAR\left( ceiling \right)$^$CLEAR\left( c \right)$^$BOX\left( box \right)$^$BANANA\left( banana \right)$

1.3 定义从初始状态到目标状态的转化

1) $WALK\left( m,n \right)$：表示猴子从$m$走到$n$。
其中个体域：$m,n\in \left\{ a,b,c \right\}$。

2) $CARRY\left( s,r \right)$：表示猴子在$r$处拿到$s$。
其中个体域：$r\in \left\{ c,ceiling \right\}$，$s\in \left\{ box,banana \right\}$。

3) $CLIMB\left( u,b \right)$：表示猴子在$b$处爬上$u$。
显然，上述操作可以分别用条件和动作来表示，为简化解题步骤不再赘述。

1.4 定义行动计划

1) $AT\left( monkey,a \right)$^$EMPTY\left( monkey \right)$^$ON\left( box,c \right)$^$ON\left( banana,ceiling \right)$^$CLEAR\left( b \right)$^$BOX\left( box \right)$^$BANANA\left( banana \right)$
替换状态：$WALK\left( a,c \right)$得状态2)

2) $AT\left( monkey,c \right)$^$EMPTY\left( monkey \right)$^$ON\left( box,c \right)$^$ON\left( banana,ceiling \right)$^$CLEAR\left( b \right)$^$BOX\left( box \right)$^$BANANA\left( banana \right)$
替换状态：$CARRY\left( c,box \right)$得状态3)

3) $AT\left( monkey,c \right)$^$HOLD\left( monkey,box \right)$^$ON\left( banana,ceiling \right)$^$CLEAR\left( b \right)$^$CLEAR\left( c \right)$^$BOX\left( box \right)$^$BANANA\left( banana \right)$
替换状态：$WALK\left( c,b \right)$得状态4)

4) $AT\left( monkey,b \right)$^$HOLD\left( monkey,box \right)$^$ON\left( banana,ceiling \right)$^$CLEAR\left( b \right)$^$CLEAR\left( c \right)$^$BOX\left( box \right)$^$BANANA\left( banana \right)$
替换状态：$CLIMB\left( box,b \right)$得状态5)

5) $AT\left( monkey,b \right)$^$EMPTY\left( monkey \right)$^$ON\left( box,b \right)$^$ON\left( banana,ceiling \right)$^$CLEAR\left( c \right)$^$BOX\left( box \right)$^$BANANA\left( banana \right)$
替换状态：$CARRY\left( banana,ceiling \right)$得状态6)

6) $AT\left( monkey,a \right)$^$HOLD\left( monkey,banana \right)$^$ON\left( box,b \right)$^$CLEAR\left( ceiling \right)$^$CLEAR\left( c \right)$^$BOX\left( box \right)$^$BANANA\left( banana \right)$
得到目标状态

则行动计划为
$WALK\left( a,c \right)$->$CARRY\left( c,box \right)$->$WALK\left( c,b \right)$->$CLIMB\left( box,b \right)$->$CARRY\left( banana,ceiling \right)$

2 实验结论

  该实验定义了描述状态的谓词，列出了猴子摘香蕉这一问题的初始化状态和目标状态。同时还分析了猴子从图示的初始状态到达目标状态，即猴子拿到了香蕉，站在箱子上，箱子位于位置b的行动计划。
